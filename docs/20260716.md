### 2.3 后端 AI 端点（Tasks #52–55）

**文件**：`controller/studio.go`（新增 ~200 行）、`router/api-router.go`（3 条新路由）

#### 新增 3 个 API 端点

| 端点 | Handler | 用途 |
|------|---------|------|
| `POST /projects/:id/quick-generate` | `StudioQuickGenerate` | 快速生成：image / analyze / describe |
| `POST /projects/:id/shots/:shotId/generate` | `StudioShotGenerate` | 单镜头生成：image 或 video |
| `POST /projects/:id/agent-create` | `StudioAgentCreate` | 创建 Hermes 任务，激活指定 skill |

#### 实现模式

- **Synthetic HTTP request**：复用 `executeHermesCompletionTask` 模式，构造 `httptest.NewRequest` 调内部 relay 端点 `/pg/chat/completions`、`/pg/images/generations`
- **异步生成**：image/video 创建 `ImagePlaygroundHistory` / `VideoPlaygroundHistory` 后启 goroutine，完成后回写 shot 的 `image_url` / `video_url`
- **Agent 创建**：构建 `HermesExecutionTask`，注入 `studioSkillSystemPrompt(skill)` 作为 system 消息，`go runHermesExecutionTask(taskId)` 异步执行

#### 新增辅助函数

| 函数 | 说明 |
|------|------|
| `executeStudioChatCompletion` | 构造 synthetic request 调 `/pg/chat/completions` |
| `extractChatCompletionContent` | 解析 OpenAI chat completion 响应提取文本 |
| `studioWritebackImageToShot` | 图像生成完成后回写 shot.image_url |
| `studioWritebackVideoToShot` | 视频生成完成后回写 shot.video_url |
| `studioQuickGenSystemPrompt` | 按 stage 返回 analyze/describe 的 system prompt |
| `studioSkillSystemPrompt` | 按 skill name 返回 6 种 Agent skill 的 system prompt |

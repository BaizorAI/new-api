# plan20260715B — Film Studio 视频生成（I2V + T2V）

## 背景

sulphur2 (`huayu-drama-4`) 现已支持 I2V (Image-to-Video) workflow。Film Studio 的 `video_gen` 阶段数据层已就绪（`video_prompt`, `video_url`, `video_task_id`, `status`），但前端没有视频生成 hook 和 UI。

### sulphur2 API 概要

```
POST /v1/videos/generations
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `prompt` | string | **必需**。视频描述文本 |
| `model` | string | `huayu-drama-4` / `huayu-drama-4-fast` / `huayu-drama-4-quality` |
| `size` | string | `512x768` (默认)、`768x1280` 等 |
| `negative_prompt` | string | 负面提示词 |
| `num_frames` | int | 帧数，默认 97（~4 秒 @24fps） |
| `fps` | int | 默认 24 |
| `num_inference_steps` | int | 默认 50 |
| `guidance_scale` | float | 默认 3.6 |
| `seed` | int | 随机种子 |
| `enhance_prompt` | bool | 是否 LLM 增强提示词 |
| `async` | bool | `true` → 202 + `{id, status}` 异步模式 |
| **`image_url`** | string | **I2V 新增**。base64 / data URL / HTTP URL，提供首帧图片 |
| **`image_strength`** | float | **I2V 新增**。0.0–1.0，首帧图片影响强度，默认 1.0 |

异步轮询：`GET /v1/videos/generations/<job_id>` → `queued` → `processing` → `completed` (含 `data[].b64_json`) / `failed`

### 前端现有架构

- **video-playground** 功能模块（`features/video-playground/`）：
  - `submitVideoGeneration(params)` → `POST /api/user/video-playground/generate`
  - `getVideoHistory(page, pageSize)` → 轮询 status（每 10s）
  - `VIDEO_STATUS`: PENDING(1) / COMPLETED(2) / FAILED(3)
  - ⚠️ 当前 `submitVideoGeneration` **不接受** `image_url`、`image_strength` 参数

- **film-studio** 功能模块：
  - `StudioShot` 类型有 `video_prompt`, `video_url`, `video_task_id`, `status`
  - `updateStudioShot(projectId, shotId, { video_url, video_task_id, status })` 已可用
  - `use-shot-image-gen.ts` 已实现图片生成轮询（可参考模式）
  - `video_gen` 阶段的 UI 已展示镜头列表，但无生成按钮

## 实施步骤

### Step 1: 扩展 video-playground API 支持 I2V 参数

**修改文件**: `features/video-playground/api.ts`、`features/video-playground/types.ts`

在 `submitVideoGeneration` 参数类型中新增可选字段：

```typescript
// api.ts — submitVideoGeneration params 增加:
image_url?: string      // I2V：首帧图片（HTTP URL / base64 / data URL）
image_strength?: number // I2V：图片影响强度 0.0–1.0
```

在 `VideoGenerationRequest` 类型中同步新增：

```typescript
// types.ts — VideoGenerationRequest 增加:
image_url?: string
image_strength?: number
```

### Step 2: 新建 `use-shot-video-gen.ts` hook

**新建文件**: `features/film-studio/hooks/use-shot-video-gen.ts`

Hook: `useShotVideoGen({ projectId, styleDna?, model? })`

流程（I2V 优先，无图片时降级 T2V）：

1. 构造 prompt：`shot.video_prompt || shot.image_prompt || shot.description`，拼接项目 `style_dna`
2. 判断是否走 I2V：如果 `shot.image_url` 存在，则附带 `image_url` 和 `image_strength`
3. 调用 `submitVideoGeneration(params)` → 获取 pending 记录
4. 将返回的 `pending.id` 写入 shot 的 `video_task_id`：`updateStudioShot(projectId, shotId, { video_task_id, status: 1 })`
5. 轮询 `getVideoHistory(1, 10)`，每 10 秒检查一次（视频生成比图片慢，用更长间隔），最多 360 次（1 小时）
6. 完成时：`updateStudioShot(projectId, shotId, { video_url, status: 2 })`
7. 失败时：`updateStudioShot(projectId, shotId, { status: 3 })`，toast 错误信息
8. Invalidate `STUDIO_QUERY_KEYS.shots(projectId)`

返回：`{ generateVideo(shot: StudioShot): void, generatingIds: Set<number> }`

参考：`use-shot-image-gen.ts` 的轮询模式，但调用 video-playground API 而非 image-playground API。

### Step 3: 修改 `studio-stage-detail.tsx` 添加视频生成按钮

**修改文件**: `features/film-studio/components/studio-stage-detail.tsx`

在 `video_gen` 阶段的镜头卡片中：

- 如果 `shot.image_url` 存在但无 `video_url`：显示 **"Generate Video"** 按钮（`Video` 图标），提示走 I2V
- 如果无 `image_url` 也无 `video_url`：显示 **"Generate Video"** 按钮（`Video` 图标），提示走 T2V
- 正在生成时（`generatingIds.has(shot.id)`）：`Loader2` 旋转 + "Generating..."
- 已有 `video_url`：显示视频缩略图/播放按钮 + hover 时"重新生成"小按钮

Shot 卡片布局调整：
```
[S1-1] [描述文本]            [图片缩略图] [视频播放/生成按钮] [⋯菜单]
```

- 如果有 `image_url`，始终显示图片缩略图（作为参考帧）
- 视频按钮独立在图片旁边

### Step 4: 视频预览组件

在 shot 卡片中，已有 `video_url` 时：
- 简单方案：`<video>` 标签 + poster（`image_url`），点击播放
- 不需要复杂播放器，一个内联 `<video>` 足够

### Step 5: shot drawer 增加 `video_prompt` 字段

**修改文件**: `features/film-studio/components/studio-shot-mutate-drawer.tsx`

在 `image_prompt` 字段之后新增 `video_prompt` textarea：
- 标签：`Video Prompt`
- placeholder：`Describe the motion and action for video generation...`
- 最大 1000 字符

## 工作流程图

```
Film Studio video_gen 阶段
│
├─ shot 有 image_url？
│   ├─ YES → I2V 流程
│   │   ├─ submitVideoGeneration({ prompt, image_url: shot.image_url, image_strength: 0.85, ... })
│   │   ├─ 轮询 getVideoHistory → VIDEO_STATUS.COMPLETED
│   │   └─ updateStudioShot({ video_url, status: 2 })
│   │
│   └─ NO → T2V 流程
│       ├─ submitVideoGeneration({ prompt, ... })  // 纯文本到视频
│       ├─ 轮询 getVideoHistory → VIDEO_STATUS.COMPLETED
│       └─ updateStudioShot({ video_url, status: 2 })
│
└─ UI 展示
    ├─ 无 video_url → "Generate Video" 按钮
    ├─ 生成中 → Loader2 旋转
    └─ 有 video_url → <video> 播放器 + 重新生成按钮
```
